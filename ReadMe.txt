Install:
 - Create a new folder in your resources folder named `DoorsM`
 - Extract the archive's contents to your "DoorsM" folder.
 - ensure "DoorsM"resource in your "server.cfg".

Command default:
 - Doors [door](Trunk, Hood, LeftFront, RightFront, LeftRear, RightRear)

Control the output of a message about opening or closing a door:
 -Open the client.lua file and in showDoorMessages it changes to the desired one(true,false)
